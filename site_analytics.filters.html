class site_analytics.filters.RequestFilter(data=None, queryset=None, prefix=None, strict=None)[source]¶

Bases: django_filters.filterset.FilterSet

Set of filters for the Request model.

The site parameter can be used to filter requests that have a url within the site (scheme & authority) argument.

The timestamp_0 parameter can be used to filter requests that were made on or after the argument. The timestamp_1 parameter can be used to filter requests that were made on or before the argument. Values for each of these parameters must be in ISO-8601 date/time format.

The username parameter can be used to filter requests that were made by the user identified by the argument.

The ip_address parameter can be used to filter requests that were made from the specified IP address.

The state parameter can be used to filter requests that were made from the specified state (two-character upper-case abbreviation).
